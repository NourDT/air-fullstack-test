# Frontend

## Overview

- Languages &amp; technologies
    - React
    - Tailwind CSS
    - Typescript
    - Dayjs
    - Jest

## Setup

Install required node packages
```
$ npm install
```

Substitute `<URL>` with endpoint URL from backend deployment.

```
$ echo "REACT_APP_ENDPOINT_URL=<URL>" > .env
```

## Running the app

```
$ npm run start
```
